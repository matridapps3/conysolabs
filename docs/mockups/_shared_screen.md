# Mockup screen — what's on it

All three mockups show the **same representative Bench screen** so the comparison is purely about visual identity.

**Screen:** Result view of a *Process Capability* analysis on `cycle_time_minutes`.

Contains:
- Top brand area (logo + product name + nav back to Conyso)
- Left sidebar with the analysis tree (Datasets · Analyses · Tools · Recipes)
- Main panel:
  - Page title + dataset breadcrumb
  - Histogram with spec lines (LSL / USL / target)
  - Metrics card (Cp · Cpk · Pp · Ppk · Z-bench · % out-of-spec)
  - "What this means" plain-English interpretation
  - "Next steps" action plan (rule-based suggestions)
- Footer

**Files:**
- `a-conyso-premium.html` — fully matches conyso.com (dark, sparse, premium)
- `b-bill-blue.html` — current Bill-style light theme (no change)
- `c-hybrid.html` — Conyso dark chrome wrapping light working content
